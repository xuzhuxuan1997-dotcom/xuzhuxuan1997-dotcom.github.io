# 网站内容编辑指南

## 📁 文件结构

```
src/
├── config.ts    ← 所有内容配置在这里修改
└── App.tsx      ← 代码逻辑（一般不需要修改）
public/
└── images/      ← 放置你的图片
```

## ✏️ 如何修改文字内容

打开 `src/config.ts` 文件，直接修改对应的内容即可：

### 基本信息
```typescript
export const SITE_CONFIG = {
  title: "NP.",           // 网站标题/Logo
  author: "你的名字",    // 作者名
  email: "your-email@example.com",  // 邮箱地址
  copyright: "© 2026 你的名字",      // 版权信息
};
```

### 导航栏
```typescript
export const NAV_ITEMS = [
  { label: "作品", href: "work" },
  { label: "关于", href: "about" },
  { label: "联系", href: "contact" },
];
```

### 主视觉区域
```typescript
export const HERO_CONFIG = {
  title: "你的标语",
  subtitle: "你的个人介绍",
  ctaButton: "打个招呼",
};
```

### 作品项目
```typescript
export const PROJECTS = [
  {
    id: "project-1",
    title: "项目名称",
    category: "项目分类",
    description: "项目描述",

    link: "#",
    featured: true,
  },
  // 添加更多项目...
];
```

### 关于区域

## 🖼️ 如何添加/更换图片

### 1. 添加图片
将图片文件复制到 `public/images/` 目录下

### 2. 在项目中使用图片
在 `PROJECTS` 数组中找到对应的项目，修改 `image` 字段：

```typescript
image: "/images/your-image.jpg"
```

### 3. 如果不想用图片，只用背景色
将 `image` 设为空，并设置 `color`：

```typescript
image: "",
color: "bg-gradient-to-br from-blue-400 to-purple-600"
```

## 🎨 背景色可选值

- `bg-gradient-to-br from-blue-400 to-blue-600` - 蓝色渐变
- `bg-gradient-to-br from-purple-400 to-purple-600` - 紫色渐变
- `bg-gradient-to-br from-pink-400 to-pink-600` - 粉色渐变
- `bg-gradient-to-br from-green-400 to-green-600` - 绿色渐变
- `bg-gradient-to-br from-orange-400 to-orange-600` - 橙色渐变
- `bg-gradient-to-br from-red-400 to-red-600` - 红色渐变

## 🚀 重新查看效果

保存修改后，浏览器会自动刷新显示最新效果。
